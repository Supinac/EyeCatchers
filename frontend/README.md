# Autism React + Pixi Starter

This starter uses **React for screens** and **PixiJS only inside the game area**.

## Run

```bash
cd frontend
npm install
npm run dev
```

Open the Vite URL shown in terminal.

## What is inside

- Black background / white UI
- Calm layout
- Home page with 6 games
- Game config page
- Play page
- Result page
- Settings page
- One working Pixi mini-game: **Find Circle**
- Mock API layer in `src/api`
